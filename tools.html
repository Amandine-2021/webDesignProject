Front-end Frameworks, Libraries and Tools

Bootstrap

is one of the most popular design framework in the world. Bootstrap is an HTML, CSS and Javascript framework that allows you to create clean-looking websites and provides pre-built code that solves many of the tricky elements, such as responsiveness. Check the W3Schools tutorial.

ReactJS

is a frontend JavaScript library that follows a component-based website development model.Created by Facebook, it allows you to build powerful, fast, user-friendly, and reactive web apps.

Angular

is a popular JavaScript front-end web development framework (developed by Google), designed specifically for developing dynamic web applications. With this framework, you can easily create front-end based applications without needing to use other frameworks or plugins.

JQuery

is a widely-used, lightweight JavaScript library. JQuery is a reusable piece of code that allows its users to easily work with the DOM elements, and handle events. As its motto says, JQuery enables the developer to "write less, do more".

Adobe

- we also need to mention Adobe software (paying subscription needed) that are useful for web design: Adobe Photoshop is one of the best imaging and graphic design software that provides great photo editing tools. Adobe Illustrator is a standard vector graphics software used by web designers to create web logos and icons. Dreamweaver is a web design software that allows you to quickly create and publish web pages.

WordPress

is a free and open-source content management system (CMS) written in PHP and paired with a MySQL or MariaDB database. Originally created as a blog-publishing system, WordpPress now support other types of web content. WordPress is one of the most popular content management system.

Back-end Frameworks and Tools

NodeJS

is an open-source, cross-platform JavaScript runtime environment that allows you to write server-side applications in JavaScript. It executes JavaScript code on your machine instead of in the browser.

Express

is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

Django

is a high-level Python Web framework that encourages rapid development and clean, pragmatic design. It makes it easier to build better Web apps more quickly and with less code.

Ruby on Rails

is an open source software. Rails is a web application development framework written in the Ruby programming language, designed to make programming web applications easier.

PHP

is a server scripting language, and a powerful tool for making dynamic and interactive Web pages.

Other Tools

GitHub

, a widely used version-control and collaboration platform. It is a source code repository that web developers use to share their projects or contribute to projects making collaboration on projects efficient. It allows developers to host and review code, manage projects, and build software.
