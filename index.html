Welcome to Learning Web Development

Learning how to program and develop websites is an interesting skill to develop in our technology-driven world. While it might be intimidating at first, learning how to code is a very rewarding experience. With hard work, motivation and perseverance, anyone can learn the skills needed to develop beautiful and usable websites. The new set of skills that you learn along the way might take you on a new journey in the web development world. Learning how to code is a first step on this path.

As mentioned earlier, your first steps in learning web development might be difficult and intimidating. This is okay: learning is a lifelong journey. With the amount of information online, it's easy to feel overwhelmed by the vast quantity of resources and anyone can face a choice overload: not being able to make a decision when faced with too many options. While this is a reality, we should not feel discouraged by it. The amount of resources is actually something we should be thankful for. Different resources might fit different learners' styles and everyone should be able to find educational tools that fit their way of learning. We are so far from the times when you just add a few paid resources available to learn new skills. Let's embrace this.

This website is there to help you find educational resources in web development. It's an overview of some of the best resources to start on your self-teaching adventure. We'll present some of the best books on the topic as well as a lot of free web resources and a few paying classes.

Before you start

Before you start on your learning journey, let's have a quick talk about your front-end web development set-up.

  • An operating system: it could be MacOS, Windows or Linux. You should pick the OS you are comfortable with.
  • A browser: being able to see how your site looks on different browsers is a must in web development. Having Chrome, Firefox, Internet Explorer, Safari, Edge and Opera installed might help you see yours sites in different browsers. You should also use cross browser testing tools to test compatibility with older browsers.
  • A text editor. There are so many options out there, my favorites are: Brackets, SublimeText, Atom, VisualStudioCode. Download them, play around and see which one you prefer using.
  • Another great resource for beginner is CodePen: an easy to use web development environment to build, test, and discover front-end code. It helps developers to better understand how the website is built and how code works.
