Free web development classes

Codecademy

is an online learning site that provides a very hands-on approach to learning how to code. They have free classes in HTML, CSS and JavaScript. They also offer paying classes to take your learning to the next level. Codecademy's approach is a “learn by doing” approach. As you progress through the lessons, you need to type code as you learn.
  • The pros: it's free, it's easy for a beginner as you do not need to set up any environment, it offers classes in many languages.
  • The cons: it does not teach you to think like a programmer, it focuses on how to write the syntax but not on developing problem-solving skills that are essential for a programmer.
Great tool for beginning your coding journey!

Free Code Camp

is an extensive free online resource that covers every web topics from front-end to back-end. Kind of an online coding bootcamp! It's interactive with plenty of exercises to practice your skills. It's comprehensive: from front-end topics to back-end development.
  • The pros: it's completely free, it covers all the different topics of web development.
  • The cons: not many!
Best tool to take you from beginner to the next level! Best free resource!

The Odin Project

is another amazing free open source coding curriculum that can be taken entirely online. It is different from other online programs. It pulls the best resources from around the web and compiles them to create a complete curriculum. It is a compilation of the best learning resources available already prepared for you
  • The pros: it is a comprehensive ready-to-use curriculum. No need to go check different resources. Just follow their lessons step by step and complete the projects to improve your skills.
  • The cons: not many!
One of the best tool if you do not have time to compare different platforms. The best learning tools in one place!

W3Schools

offer free tutorials in all things web. Their tutorials are easy to follow and allow you to quickly learn the basics on a new topic. It covers a wide variety of web development languages and frameworks.
  • The pros: it's free and its focuses only on web development! It has a lot of quick references pages.
  • The cons: it does not always go in the details. It uses the name W3 in its name even though it is not affiliated with the W3 Consortium (that can be a little misleading).
Amazing resource for learning a new language and for quick references.

References

MDN Web Docs aka as the Mozilla Developer Network

is the reference in web development. It's great for beginners but also for seasoned web developers who needs to review or sharpen their skills.
  • The pros: it's comprehensive, very detailed and up to date. It's full of coding examples, showing you how to use elements, attributes, properties... in well written code. It's the ultimate reference resource for web developers.
  • The cons: none!
A must use for quick references and for in-depth learning. You will always find something new to learn. Best, most comprehensive and up to date reference resource!
