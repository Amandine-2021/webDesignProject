Books to Read

Here is a list of some interesting books to read to learn about JavaScript.

Eloquent JavaScript: A Modern Introduction to Programming by Marijn Haverbeke

This book is my favorite to learn JavaScript. Great introduction to programming. It teaches you important programming concepts through the use of JavaScript. It's full of exercises to practice. Not an easy book if you do not have any programming background but it will teach you the fundamentals of programming in a clear, concise way.

Buy Eloquent JavaScript by Marijn Haverbeke or access the book for free on the Eloquent JavaScript website

JavaScript and JQuery: Interactive Front-End Web Development by John Duckett

This book is also very popular: it's the perfect book to learn web design, it's a complete course with examples, and a lot of hands-on practice! It's great for beginners but will also get you to more advanced topics fast, allowing the more experienced web designers to update their skills. Must read!

Buy JavaScript and JQuery by John Duckett or check the author's website

A Smarter Way to Learn JavaScript by Mark Myers

A different and innovative way to learn Javascript. This book has short chapters in which you learn important concepts. After each chapter, you get to practice online with a serie of questions and exercises that help you retain the knowledge. Unique learning experience for beginners.

Buy A Smarter Way to Learn JavaScript by Mark Myers and check the book's online exercises

Head First HTML5 Programming by Elisabeth Robson and Eric Freeman

Despite its title, this book is mostly about JavaScript and how to make pages dynamic through the use of HTML5 combined with JavaScript. It explains the relation between the DOM (Doocument Object Model), the HTML, and the JavaScript. Fun and humorous book that is a great introduction to creating web apps.

Head First HTML5 Programming by Elisabeth Robson and Eric Freeman

JS Sandbox

Sandbox tools allow you to quickly test your code without having to set up and save HTML, CSS and JavaScript documents simply to test and debug small code snippets. You can play, debug and experiment with with these free sandbox tools: JSFiddle, CodePen, CodeSandbox, PlayCode.
