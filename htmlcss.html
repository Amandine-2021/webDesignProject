Free web resources for learning HTML and CSS

First, be sure to check the General Courses about web development.

Some other great resources for HTML and CSS are:

Learn to Code HTML and CSS with Shay Howe

is an interactive beginner’s guide with the goal to teach you how to develop and style websites with HTML and CSS. It covers all the common elements of front-end design and development.

CSS Tricks

is an amazing website about building websites. Despite its title, its content is about everything related to web development, not just CSS. The multiple guides are in-depth and go into details well beyond beginner level on several web development subjects like grid, flexboxes, color, and more... The website offers many articles by guest writers, videos and references on specialized topics. Its content is always evolving. Must check!

Interneting is Hard

is a beautiful web development tutorial webite. It explains concepts in a clean and simple manner with beautiful diagrams that help you remember concepts in a visual way. Great content, very aesthetic, easy to follow and clear!

Don't fear the Internet

is another nice beginners’ guide to the fundamentals of web design. Its audience is non web designers making it very beginner friendly. Easy to understand videos, Don't Fear the Internet is a good place to start understanding HTML and CSS coding.

MDN Web Docs (aka Mozilla Developer Network)

is a comprehensive learning platform that provides tutorials as well as current web standards. You will find in-depth HTML tutorials, CSS tutorials, and many other references. This is the place to go if you need to learn something new or quickly check how to use an element or an attribute.

W3schools

is another widely used reference websites for html and css. It offers an html tutorial, a css tutorial, and many references: a list of html tags, common characters sets, a css selector reference, and much more. Their tutorials are easy to follow and great for beginners.

Smashing Magazine

is an online magazine for topics in web design and web development. It's a great professional resource with articles on a wide range of web related topics. This company also publishes e-books. One of world's most popular and highly regarded magazines in the area of web design and web development. Check it out!

Books to Read

Here is a list of some interesting books to read to learn about HTML

HTML & CSS design and build websites by John Duckett

This book is very popular for many reasons: the content is amazing, easy to grasp. It is not a traditional textbook, it's visual, colorful, and well organized. Its use of graphics and images makes the book very engaging. The author introduces the different elements side by side with concrete examples. Wonderful book that can be used as an how-to and as a reference later on. It's also affordable. Definitely a must have!

Buy HTM & CSS by John Duckett or check the author's website

Learning Web Design: A Beginner's Guide to HTML, CSS, JavaScript, and Web Graphics by Jennifer Robbins

This book is also very popular: it's the perfect book to learn web design, it's a complete course with examples, and a lot of hands-on practice! It's great for beginners but will also get you to more advanced topics fast, allowing the more experienced web designers to update their skills. Must read!

Buy Learning Web Design by Jennifer Robbins or check the author's website

HTML5 and CSS3 by Patrick Carey

This book was used in a class at my local community college, this is the only reason why I am adding it here. It's a textbook with a lot of step-by-step tutorials. If you are looking for a textbook tutorial that will guide you through every steps, this could be an option. A lot of website examples with a hands-on approach.

Buy HTML5 and CSS3 by Patrick Carey

Specific HTML and CSS Topics

HTML Elements

HTML consists of a series of elements, which you use to enclose or wrap, different parts of the content to give it some meaning. Elements consist of an opening tag, a closing tag and some content. To quickly see all the different HTML elements available, check MDN HTML elements reference as well as the MDN HTML attribute reference.

Design Inspiration

CSS (Cascading Style Sheets) describes how HTML elements are displayed. CSS allows you to use your imagination and your design skills to style the content of your projects. If you need inspiration, explore CSS Zen Garden, a creative resource "built to demonstrate what can be accomplished visually through CSS-based design", by showcasing a multitude of inspiring designs based on the same single HTML file. For more inspiration, check Awwwards, CSS Design Awards, and Dribble.

Layout

Read this article for basic understanding of layout in CSS: Getting Started With CSS Layout, and explore this website: Learn CSS Layout, a great tutorial that teaches the fundamentals of CSS Layout.

Flexbox

Start with the CSSTricks Guide to Flexbox, a comprehensive, visual guide that helps you understand the flexbox concept. For practicing the flexbox layout, play Flexbox Froggy, an adorable and fun game.

CSS Grids

Read the W3Schools Grid Tutorial, the CSS Tricks complete guide to Grid as well as this article on best grid layout practices. For a more hands-on approach, try:

  • CSS Grid Garden - a fun and cute game to practice CSS grids.
  • Layoutit - an interactive CSS grid generator.
  • Griddy - another interactive CSS grid generator.

Fonts

The importance of chosing fonts that are relevant to your content and readable is a must in web design. Read this articles on typography: a guide to web typography and 8 simple typography tips for your designs for a quick introduction to typography. Then experiment with different fonts with Google Fonts. Google Fonts offer an amazing collection of open source designer web fonts that can easily be integrated into your design projects.

Colors

In website design, colors really matter. Choosing the right website color scheme is an essential part of designing a website.

First, check the W3Schools Color Tutorial - a very comprehensive color tutorial that explains how to pick color values, color schemes and basic color theory.

There are many tools to help you select the right colors:

  • Adobe Color - a free and one of the best tool to help you generate a color palette.
  • Paletton - a powerful tool to create color combinations and palettes of colors.
  • Coolors - a smart color scheme generator.
  • Colors by names - a quick way to pick a color by name for quick testing (but limited to 140 colors only).
  • Don't forget to check the Contrast Checker to help you find a color that meets the desired level of contrast.

Color Gradients

CSS gradients let you display smooth transitions between two or more colors. It allows you to add beautiful color effects in your design. Read this great explanation of CSS Gradients.

Tools for generating gradients:

  • CSS Gradient - an easy to use tool to make beautiful gradients.
  • Eggradients - an inspiring and humorous tool (check the gradients' titles!) for generating gradients.

Images

Adding images to your websites is a great way to add color, visual clues and meaning to your content. Here are a few websites where you can find free images & royalty free stock photos:

  • FreeImages
  • Unsplash
  • Pixabay
  • Pexels

Validation

To end, let's mention the importance of validating your html and css code as well as checking the color contrast of your design.

  • HTML Validator - the W3C markup validator.
  • CSS Validator - the W3C CSS validator.
  • Contrast Checker - contrast checker for colors.
  • CanIUse - to check web compatibility across browsers.
